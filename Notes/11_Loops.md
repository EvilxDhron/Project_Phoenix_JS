# Loops in JavaScript

## Q.1 What is a Loop?

A loop is a control structure that repeatedly executes a block of code until a specified condition becomes false. It helps eliminate repetitive code and makes programs efficient. Loops save time, reduce code repetition, and make programs dynamic.

## Q.2 How many types of Loops are in JavaScript?

JavaScript provides five main loops :-

    1. for loop
    2. while loop
    3. do...while loop
    4. for...of loop
    5. for...in loop

## Q.3 What is the difference between while and do...while loop?

- while loop
    - Checks condition first.
    - May execute zero times.

- do...while loop
    - Executes first.
    - Checks later.
    - Runs at least once.

## Q.4 What is the difference between `for` and `while`?

A `for` loops is preferred when the number of iterations is known because initialization, condition, and update are written in one place. A `while` loop is preferred when the number of iterations is unknown and depends on a condition during execution.

## Q.5 What does `break` do?

The `break` statement immediately terminates the nearest enclosing loop or `switch` statement and transfers control to the statement following it.

## Q.6 What does `continue` do?

The `continue` statement skips the rest of the current iteration and moves directly to the next iteration of the loop.
