# Arrays in JavaScript

## Q.1 What is an Array?

An array is a special data structure used to store multiple values in a variable.

OR

An array is an ordered collection of values stored in a single variable. Each element is accessed using its index, starting from 0.

Ex :-

    let fruits = ["apple", "banana", "mango"];

## Q.2 How to declare an Array?

There are two methods to declare arrays :-

    - Array literal
        let arr = [1, 2, 3];

    - Array Constructor
        let arr = new Array(1, 2, 3);

## Q.3 Why do we use Arrays?

Arrays are used to store multiple related values efficiently and to make operations like traversal, searching, and transformation easier.

## Q.4 What is the difference between `push()` and `pop()`?

`push()` adds element to the end of an Array, while `pop()` removes the last element.

## Q.5 What is the difference between `shift()` and `unshift()`?

`shift()` removes the first element, and `unshift()` adds elements at the beginning.
